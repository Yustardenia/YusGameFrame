YusTweenSystem Inspector 友好 · 轻量可拼装

推荐入口统一为 YusTween（写代码） + YusTweenSequencePlayer（策划/美术在 Inspector 里拼动效）。

推荐入口

  • 代码：直接用 YusTween.*（统一 SetUpdate / SetLink / SetId 等默认行为）
  • Inspector：挂 YusTweenSequencePlayer，用多轨道拼装 Move/Scale/Rotate/Color/Fade
目标：让“常见动效”尽量不写代码、可复用、可随 Prefab 走；代码只处理少数特殊逻辑。

YusTweenSequencePlayer 快速上手

  1. 给物体挂 YusTweenSequencePlayer
  2. 点一次 自动绑定目标（可选，但强烈建议）
  3. 点 + 移动/+ 缩放/+ 淡入淡出/+ 颜色 添加轨道，设置 To（需要指定起点就勾 指定起点（From））
  4. 把 播放时机 设为 OnEnable/Start，或在 UnityEvent 里调用 PlayForward()
播放预览：进入 Play Mode 后，Inspector 底部会出现 正向播放/反向播放/停止 按钮。

常见搭配（示例）

一个“弹窗出现”的常见组合：

轨道 0：CanvasGroupAlpha  From=0  To=1  duration=0.18  ease=OutQuad
轨道 1：LocalScale        From=0  To=1  duration=0.22  ease=OutBack

一个“按钮反馈”的常见组合：

轨道 0：LocalScale  Relative=true  To=(0.06, 0.06, 0)  loops=2  loopType=Yoyo
