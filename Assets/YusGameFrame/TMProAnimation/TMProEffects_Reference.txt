================================================================================
TextMeshPro 链接动画效果速查表
================================================================================

使用方法：
在 TextMeshPro 文本框中，使用 <link="效果名">要动画的文字</link> 格式。
例如：你好，<link="wave">世界</link>！

--------------------------------------------------------------------------------
1. Fungus 内置效果 (无需额外脚本)
--------------------------------------------------------------------------------

| 效果名 (ID)   | 描述                                      | 视觉感受                   |
|:--------------|:------------------------------------------|:---------------------------|
| jitter        | 字符随机剧烈抖动                          | 恐惧、寒冷、不稳定         |
| angry         | 单词级别的剧烈震动                        | 愤怒、大吼                 |
| spooky        | 缓慢的怪异扭动                            | 幽灵、诡异、神秘           |
| unknowable    | 字符级别的扭动                            | 混乱、不可名状、疯狂       |
| wave          | 经典的上下波浪运动                        | 轻松、海洋、漂浮           |
| swing         | 像钟摆一样摇晃                            | 摇摆、犹豫                 |
| bounce        | 单词上下弹跳                              | 活泼、强调                 |
| excited       | 字符快速上下跳动                          | 兴奋、高兴                 |
| glow          | 颜色在 HSV 空间脉冲变化                   | 发光、魔法、能量           |

--------------------------------------------------------------------------------
2. YusGameFrame 自定义效果 (由 CustomTMProEffects.cs 提供)
--------------------------------------------------------------------------------

| 效果名 (ID)   | 描述                                      | 视觉感受                   |
|:--------------|:------------------------------------------|:---------------------------|
| heartbeat     | 文字像心脏一样有节奏地缩放                | 心跳、紧张、生命力         |
| spin          | 字符原地旋转                              | 眩晕、旋转                 |
| rain          | 文字向下坠落的阶梯效果                    | 下雨、下坠、消极           |
| glitch        | 赛博朋克风格的故障闪烁和位移              | 电子故障、黑客、系统错误   |

--------------------------------------------------------------------------------
注意：
- 效果名区分大小写（通常全是小写）。
- 多个效果不能直接叠加在同一个 link 标签里，但可以嵌套（虽然不推荐，可能会有冲突）。
- 确保场景中有 TextMeshPro 组件，脚本会自动运行。
