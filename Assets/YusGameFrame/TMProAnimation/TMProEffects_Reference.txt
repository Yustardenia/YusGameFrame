================================================================================
TextMeshPro Link 动画效果速查表
================================================================================

使用方法：
在 TextMeshPro 文本中，使用 <link="效果名">要动画的文字</link> 格式。
例如：你好，<link="wave">世界</link>！

type（打字机）支持参数（可选）：
- <link="type:12">文本</link>             // 每秒 12 字符
- <link="type:12,0.2">文本</link>         // 每秒 12 字符，延迟 0.2 秒开始
- <link="type:12,0.2,0.05">文本</link>     // 额外设置淡入时长 0.05 秒

reveal（擦除显现）支持参数（可选）：
- <link="reveal:0.7">文本</link>           // 时长 0.7 秒
- <link="reveal:0.7,0.06">文本</link>      // 时长 0.7 秒，羽化 0.06（按本段宽度归一化）

garble（动态乱码）：
- <link="garble">原文多长，乱码就多长</link>
- 特点：不显示原文字形，只显示动态变化的乱码字符（英文字母 + 符号如 !@#$%^ 等，直接改写 TMP 文本内容）
- 如果显示成“方块/豆腐块”：说明当前 TMP Font Asset 实际不包含这些字符对应 glyph（需要在 Font Asset Creator 里把它们烘进去）。

3. Link 参数可视化（编辑器工具）
--------------------------------------------------------------------------------

把 `TMProLinkIdVisualizer` 挂到同一个 GameObject（或任意对象）上，并把 Target 指向你的 `TMP_Text`：
- Inspector 会列出当前文本里所有 `<link="...">...</link>` 的 LinkID、EffectKey、参数字符串
- Scene 视图里会给每个 link 画出范围框，并标注 EffectKey（方便检查“哪个 link 写错了”/“参数有没有生效”）

1. Fungus 内置效果（无需额外脚本）
--------------------------------------------------------------------------------

| 效果名(ID)     | 描述                                      | 视觉感受                   |
|:--------------|:------------------------------------------|:---------------------------|
| jitter        | 字符随机剧烈抖动                            | 恐惧、寒冷、不稳定         |
| angry         | 单词级别的剧烈震动                           | 愤怒、大吼                 |
| spooky        | 缓慢的怪异扭动                              | 幽灵、诡异、神秘           |
| unknowable    | 字符级别的扭动                              | 混乱、不可名状、疯狂       |
| wave          | 经典的上下波浪运动                           | 轻松、海洋、漂浮           |
| swing         | 像钟摆一样摇摆                              | 摇摆、犹豫                 |
| bounce        | 单词上下弹跳                                | 活泼、强调                 |
| excited       | 字符快速上下跳动                            | 兴奋、高兴                 |
| glow          | 颜色在 HSV 空间脉冲变化                      | 发光、魔法、能量           |

2. YusGameFrame 自定义效果（由 CustomTMProEffects.cs 提供）
--------------------------------------------------------------------------------

| 效果名(ID)     | 描述                                      | 视觉感受                   |
|:--------------|:------------------------------------------|:---------------------------|
| heartbeat     | 文字像心脏一样有节奏地缩放                   | 心跳、紧张、生命力         |
| spin          | 字符原地旋转                                | 眩晕、旋转                 |
| rain          | 文字向下坠落的阶梯效果                       | 下雨、下坠、沉落           |
| glitch        | 赛博朋克风格的故障闪烁和位移                 | 电子故障、黑客、系统错误   |
| type          | 打字机 / 逐字显示                            | 对话、剧情、慢速出字       |
| shine         | 高光扫过（扫光从左到右循环）                 | 强调、获得道具、提示       |
| flicker       | 霓虹闪烁（亮灭+轻微色偏）                    | 招牌灯、故障感、氛围        |
| float         | 轻微漂浮 + 呼吸缩放                          | 梦境、魔法、轻提示         |
| impact        | 入场“砰”一下（一次性冲击回弹）               | 爆点台词、惊讶、强调        |
| reveal        | 擦除显现（按位置从左到右）                    | UI 显示、标题出现           |
| decode        | 乱码→原文（会改写 TMP 文本内容）             | 解码、黑客、系统提示        |
<link="garble">原文多长，乱码就多长</link>
--------------------------------------------------------------------------------
注意：
- 效果名区分大小写（通常全是小写）。
- 多个效果不要写在同一个 link 标签里；如需组合请嵌套（不推荐，可能冲突）。
- 运行时需要 TextMeshPro 组件启用 Rich Text（默认开启），且文本里确实包含 <link> 标签。
